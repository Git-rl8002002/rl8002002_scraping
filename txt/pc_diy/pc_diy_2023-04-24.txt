趨勢科技加入Microsoft私有邊緣運算生態系 這項合作將帶動在Azure上安全地建置5G專網的普及
全球網路資安解決方案領導廠商趨勢科技(東京證券交易所股票代碼：4704) 在 2023 年德國漢諾威展 (Hannover Messe 2023) 上宣布該公司已獲選為 Microsoft Azure 私有 5G 多接取邊緣運算 (MEC) 軟體合作夥伴。



如需本消息的進一步詳情，請至：https://azure.microsoft.com/en-us/solutions/private-multi-access-edge-compute-mec/

趨勢科技營運長Kevin Simzer表示：「趨勢科技超過 25 年來一直是 Microsoft 的合作夥伴，我們很高興能將彼此的合作關係拓展至快速興起的 5G 網路領域。我們 CTOne 團隊所開發的功能將協助在 Microsoft Azure Private 5G Core 上建置部署環境的企業加速法規遵循作業，降低網路資安風險。」

企業 5G 專網能為各式各樣的企業 (如醫療機構) 與智慧製造業者帶來工廠、園區及其他據點的轉型契機，提供低延遲、高頻寬的連線。這是一個規模已將近 14 億美元的市場，並且預計將以 49% 的年複合成長率一路成長至 2030 年。

儘管 5G 專網已經比公共 5G 網路更加安全，然而這類部署環境所帶來的大量雲端技術、IoT 端點以及開放原始碼軟體 (如 O-RAN 部署環境) 將大幅增加企業的受攻擊面。研究指出，就連行動電信業者也認為他們缺乏充分的工具和知識來防範這些疑慮。

趨勢科技憑藉著多年的產業經驗所開發出來的 Trend Micro Mobile Network Security (TMMNS) 是一套專為同時保護 5G 專網與端點而設計的聯合防護，能簡化法規遵循作業，強化資通訊技術 (IT/CT) 環境的整體安全。

這套解決方案將在 Azure Marketplace 上架，提供一個絕佳的選項給想要採用 Azure Private 5G Core 來安全地部署 5G 專網的 Microsoft 客戶選購。

趨勢科技是 Microsoft 策略 500 大客戶 (Strategic 500 Customer) 與全球黃金級獨立軟體廠商 (Global Gold ISV)，該公司在 2021 年宣布的一項策略中明確表示將在 Microsoft Azure 上打造雲端式防護解決方案來開創聯合銷售商機。

本次在 2023 年漢諾威展上宣布的最新消息將延續這項動能，同時將彼此的合作拓展至快速成長的 5G 專網領域，目前已有多個與共同客戶合作的專案正在進行當中。
為強化該領域的專業能力，趨勢科技日前宣布成立一家新的子公司 CTOne 來聚焦 5G 資安領域。該公司已協助台灣的 NEXCOM (新漢) 採用領先的次世代無線網路安全防護來加速智慧工廠的數位轉型。

世界地球日｜OPPO智慧雲（大灣區）數據中心，為地球增添一抹創新之「綠」 
從1970年4月22日第一次地球日活動至今，從「講好地球故事」到「投資我們的星球（invest in our planet）」，54年來，世界地球日不斷以獨特的方式提醒大眾關注環境問題。實際上，保護地球不僅在地球日這一天，OPPO將可持續發展融入企業長期發展戰略並長期投入。2023年世界行動通訊大會期間，OPPO承諾2050年實現自身營運碳中和，發展零碳數據中心建設正是OPPO實現碳中和道路上的一項重要探索。

OPPO智慧雲（大灣區）數據中心是OPPO首個自建大型數據中心，是OPPO數據安全中心、智慧計算中心、算法創新中心，也是OPPO全球混合雲基礎設施建設的關鍵組成部分。為減少數據中心能源消耗和溫室氣體排放，OPPO透過綠色基礎設施建設，以及創新科技應用，推動數據中心的綠色營運和能效提升。

提到碳足跡，你可能首先想到的是來自工業、交通、農業等領域所產生的碳排放。實際上，在科技領域，數據中心作為資料儲存、人工智慧和高效能運算等數位產業和前沿技術的重要載體，其能耗也十分巨大。為了減少數據中心用電產生的碳排放， OPPO智慧雲（大灣區）數據中心100%使用可再生能源。

2022年投入使用以來，數據中心合計使用綠電6.178千兆瓦時，相當於向地球減少了3,600噸二氧化碳排放。同時，數據中心回收雨水用於冷卻系統循環使用，每年幫助中心節省約30,000噸水資源。
 



據統計，全球數據中心在運行中共消耗了全球總發電量的3%1。如此驚人的能源消耗背後，是耗電量居高不下的冷卻系統。隨著算力需求和功耗密度的不斷增長，依靠風扇、空調等機械設備進行製冷散熱的傳統風冷散熱局限性日益明顯。因此，尋求高效的散熱方式成為業內一大痛點。

為了提升能源利用效率，降低碳排，OPPO積極研究和部署浸沒式液冷GPU伺服器集群。浸沒液冷技術，是指將伺服器直接浸沒在不導電的液體中，直接將運轉過程中產生的熱能傳導給液體，實現無機械製冷。溫度上升的液體透過循環冷卻的方式，再回流繼續吸收熱能，回收的熱能還能繼續用於公寓供暖、生活熱水提供等場景實現二次利用，極大地提高了能源的利用效率。

浸沒液冷設備的節能嘗試初見成效，節能效果提升了45%，數據中心電源使用效率（PUE）低至1.15，處於行業領先水準。

2030年全球元宇宙市場規模上看9366億美元 COMPUTEX元宇宙浪潮研討會將剖析元宇宙應用趨勢與可行商機
COMPUTEX共同主辦單位TCA（台北市電腦公會）表示，根據Grand View Research研究報告指出，預估2030年全球元宇宙市場規模上看9366億美元，2023 - 2030年均複合成長率（CAGR）達41.6％，其中又以與最終使用行業應用需求成長最快，包括媒體業、娛樂業、教育業、航太業、國防業等領域，AR、VR、MR等XR軟硬體設備產值也持續增加，並且元宇宙可透過加密貨幣技術，快速擴張市場規模且確保使用者資料安全性。



為讓產業人士更加了解元宇宙應用趨勢與可行商機，TCA將於4月26日假台北市電腦公會B1會議室，辦理「COMPUTEX系列研討會：元宇宙浪潮下各領域的機會與挑戰」，邀請宏達電、NVIDIA、資策會MIC、全球動力科技等產業專家，從元宇宙企業應用、全方位平行化元宇宙應用、元宇宙發展局勢與應用分析、元宇宙智慧雙生XR虛實整合體感實際案例等面向，進行專題報告與案例分享，讓與會人士了解能如何快速掌握元宇宙商機，以及如何建置元宇宙應用服務。

在「元宇宙發展局勢與應用分析」專題演講中，資策會MIC資深產業分析師兼專案經理柳育林，將從元宇宙發展局勢、元宇宙應用分析這兩大面向，盤點元宇宙在技術面、經濟面、社會面、市場面等商機特性，以及廠商切入元宇宙產業的真正可行方向。

由於垂直場域的元宇宙企業應用，是目前看到最實際且最可行的元宇宙商機，因此研討會邀請到宏達電SVP, Viverse Product Head的Tai Ito，介紹如何透過VIVERSE軟硬體生態系，快速建置專屬於企業應用場域的元宇宙解決方案，並且確保個人交易與資料的安全性。

NVIDIA的Omniverse是建置元宇宙各類應用當中，相當熱門的元宇宙平台，並且已經在製造業當中有多個實際導入案例，因此邀請NVIDIA開發者關係經理黃威仁博士，以實際應用案例，介紹如何使用NVIDIA Omniverse打造跨維度、全方位、平行化的元宇宙，並且具備準確物理特性、完美可視化、精準模擬並與現實同步等特性。

智慧城市是各國政府近年來努力發展的方向，也是規模最大、影響範圍最廣的垂直應用場域，為了讓產業了解元宇宙技術如何在智慧城市場域中使用，將由全球動力科技總經理陳俊男，同時也是財團法人防災產業協會常務理事、台灣虛擬及擴增實境產業協會理事，以台灣發展中的智慧城市數位雙生元宇宙、元宇宙XR技術整合介紹等面向，介紹台灣在智慧城市場域中的元宇宙應用實際成功案例，並分享相關產業商機。
